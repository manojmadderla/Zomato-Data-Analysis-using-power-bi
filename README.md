# 📊 Zomato Data Analysis Dashboard (Power BI)

## 📌 Overview

This project presents an interactive **Power BI dashboard** built using Zomato dataset to analyze restaurant trends, customer preferences, and service availability across different regions. The dashboard provides key insights into cuisine popularity, online delivery trends, and geographic distribution of restaurants.

---

## 🎯 Objectives

* Analyze restaurant distribution across countries and cities
* Identify popular cuisines and food trends
* Understand online delivery and table booking availability
* Provide insights for business decision-making in the food industry

---

## 📂 Dataset

The dataset used in this project contains information such as:

* Restaurant Name
* Location (Country, City)
* Cuisines
* Online Delivery (Yes/No)
* Table Booking (Yes/No)
* Ratings & Votes

---

## 📈 Dashboard Features

### 🔹 1. Country Filter

* Allows users to filter data by country (e.g., South Africa, Sri Lanka)

### 🔹 2. Online Delivery Analysis

* Donut charts showing:

  * % of restaurants offering online delivery
  * % of restaurants not offering online delivery

### 🔹 3. Table Booking Analysis

* Visual representation of availability of table booking services

### 🔹 4. Cuisine Distribution

* Bar chart displaying most popular cuisines:

  * Chinese
  * North Indian
  * Cafe
  * Bakery
  * Desserts

### 🔹 5. Restaurant Distribution

* Insights into how restaurants are spread across locations

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization
* **Excel / CSV Dataset** – Data source
* **DAX (Data Analysis Expressions)** – Calculations & measures

---

## 📊 Key Insights

* A majority of restaurants do **not offer online delivery**
* Popular cuisines include **Chinese and North Indian**
* Certain countries have a higher concentration of restaurants
* Online services vary significantly by region

---

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use slicers (filters) to explore different countries
3. Interact with charts to drill down into insights
